Cursor Clone Website

A simple website made using only HTML and CSS.
This project is a copy of the Cursor website design, created for practice and learning basic web development.
---------------------------------------------------------------------------------------------------------------
Project Overview

This project is a desktop-only website that looks similar to the Cursor homepage.
The main goal of this project is to understand page structure, layout, and styling using HTML and CSS only.

No advanced tools or frameworks were used.

What This Website Contains

The website is divided into different sections, just like a real product website.
--------------------------------------------------------------------------------------------
1. Top Navigation Bar

Website logo

Navigation links (Home, Features, etc.)

One main button

Dark background

2. Main Section (Top Section)

Big heading text

Short description

One button for users to click

Large image showing the product

3. Company Logos Section

A row of company logos

Shows which companies use the product

4. Feature Sections

Three separate sections

Each section has:

Text on one side

Image on the other side

Image and text position changes in each section

5. Feature Cards Section

Section title

3 to 4 cards

Each card explains one feature

6. User Reviews Section

Quotes from users

User name and role shown

7. Use Case Section

Cards with image and short text

Explains how the product can be used

8. Updates Section

List of updates

Each update has a date and short description

9. About Section

Large image

Short description about the team or product

One button

10. Final Section

Big heading text

One button at the center

11. Footer

Multiple columns

Links and basic company information
------------------------------------------------------------------------------------------------------------------------------------
Rules Followed in This Project

Only HTML and CSS are used

No JavaScript

No Tailwind CSS

No AI tools used

No animations or special effects
----------------------------------------------------------------------------------------------------------------------------------
